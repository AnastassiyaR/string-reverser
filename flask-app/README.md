# Flask String Reverser

A simple web interface for reversing strings, built with Flask.

## Requirements

- Python >= 3.8
- Flask
- transformations-anrozk (from Test PyPI)

## Installation & Running

### 1. Install the package from test.pypi.org

```bash
pip install -i https://test.pypi.org/simple/ transformations-anrozk
```

### 2. Install Flask

```bash
pip install flask
```

### 3. Run the app

```bash
python app.py
```

### 4. Open in browser

```
http://127.0.0.1:5000
```
